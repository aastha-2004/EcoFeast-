# EcoFeast
EcoFeast is a web-based platform designed to bridge the gap between food donors and beneficiaries, while intelligently managing surplus food and bio-waste. Built as a socially driven project, EcoFeast empowers NGOs, donors, and recipients to contribute to a sustainable ecosystem that minimizes food wastage and maximizes outreach.

🚀 Features

Rule-based matching engine to recommend optimal beneficiaries based on food type, expiry, quantity, and location

Donor and beneficiary portal for quick registration and tracking of food transactions

Nutrition insight system to log and suggest food types based on demographic needs

Bio-waste logging and reporting module for expired or unusable food

Central calendar and dashboard for managing all upcoming donations and pickups

🛠️ Tech Stack

Backend: Flask (Python)

Database: MySQL

Frontend: HTML, CSS, Bootstrap (optional templating)

Matching Logic: Rule-based algorithm (non-AI)

Hosting: Localhost / optional deployment on Render or PythonAnywhere

📚 About the Algorithm

Unlike machine learning or AI models, EcoFeast uses a deterministic rule-based algorithm for food matching. The logic is structured around:

Proximity between donor and beneficiary

Expiry threshold of food items

Matching food types to beneficiary needs

Filtering based on allergies or nutritional tags

This ensures transparency and traceability in the food distribution process.
